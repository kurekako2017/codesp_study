## ⚠️ 注意事项

### 1. Java版本
- ✅ **正确**: Java 11
- ❌ **错误**: Java 21 (会导致class文件版本不匹配)

### 2. 测试代码
- 当前测试代码有编译错误
- 必须使用 `-Dmaven.test.skip=true` 跳过
- 后续需要修复测试代码

### 3. DevTools
- `mvn spring-boot:run` 会触发DevTools自动重启
- 建议使用打包的JAR方式运行
- 或在开发时禁用DevTools

### 4. 数据持久化
- H2数据存储在 `./data/` 目录
- 重启应用数据不会丢失
- 如需清空数据，删除 `./data/` 目录
