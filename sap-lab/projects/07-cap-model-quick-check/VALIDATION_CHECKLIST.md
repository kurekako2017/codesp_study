## CAP 快速验证清单

### 模型验证
- [ ] 所有实体都有主键
- [ ] 关联（Association）配置正确
- [ ] 组合（Composition）配置正确
- [ ] 字段类型和长度合理
- [ ] 默认值设置正确

### 服务验证
- [ ] OData 端点正确生成
- [ ] 投影（Projection）配置正确
- [ ] 视图（View）定义合理
- [ ] 操作（Action/Function）签名正确
- [ ] 权限注解配置（如需要）

### 数据验证
- [ ] 创建测试数据
- [ ] 验证 CRUD 操作
- [ ] 测试关联和组合查询
- [ ] 验证排序和过滤
- [ ] 测试分页

### 常见问题排查
1. **模型错误** - 检查 CDS 语法
2. **服务路由** - 验证 srv.cds 配置
3. **性能问题** - 检查索引和查询优化
4. **权限问题** - 验证认证配置

### 验证命令
```bash
# 编译 CDS 模型
cds compile db

# 启动本地服务
cds run

# 验证服务元数据
curl http://localhost:4004/$metadata
```
