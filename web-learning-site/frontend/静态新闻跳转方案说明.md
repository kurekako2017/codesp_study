# 静态新闻跳转方案说明

## 为什么 news.json 会报错？

JSON 格式标准不允许出现注释（如 // 或 /* ... */），也不能混用对象和数组结构。

你在 news.json 文件中加入了注释和多余的结构，导致解析时报错：
- “JSON 中不允许有注释。”
- “预期为文件结尾。”

## 正确写法

news.json 只允许如下内容（无注释、无多余结构）：

```
[
  {
    "title": "新闻标题1",
    "date": "2026-01-15",
    "summary": "这是新闻摘要1",
    "link": "../blog/article1.html"
  },
  {
    "title": "新闻标题2",
    "date": "2026-01-10",
    "summary": "这是新闻摘要2",
    "link": "../blog/article2.md"
  },
  {
    "title": "新闻标题3",
    "date": "2026-01-05",
    "summary": "这是新闻摘要3",
    "link": "https://docs.google.com/document/d/xxx"
  }
]
```

如需保留原内容，请单独存为 news.json.bak 或文档。

## 静态新闻跳转方案（适合静态网页）

1. news.json 维护新闻列表，每条新闻有 title、date、summary、link 字段。
2. 前端 index.html 用 JS 读取 news.json，渲染新闻列表。
3. 每条新闻的标题或卡片用 `<a href="...">` 包裹，点击可直接跳转到整篇文章（支持 html/md/pdf/外链）。
4. md 文件可用 docsify、markdown-it 等工具渲染，或转为 html。

### 示例 index.html 代码片段：

```html
<!DOCTYPE html>
<html lang="zh-cn">
<head>
  <meta charset="UTF-8">
  <title>新闻列表</title>
</head>
<body>
  <h1>新闻列表</h1>
  <div id="news-list"></div>
  <script>
    fetch('news.json')
      .then(res => res.json())
      .then(newsList => {
        const container = document.getElementById('news-list');
        newsList.forEach(news => {
          const item = document.createElement('div');
          item.innerHTML = `
            <a href="${news.link}" target="_blank">${news.title}</a>
            <div>${news.date}</div>
            <div>${news.summary}</div>
          `;
          container.appendChild(item);
        });
      });
  </script>
</body>
</html>
```
